RemoveEquivalentAdjacentPoints(float)

Removes adjacent points that are identical, or equivalent (i.e. closer than a given threshold).

Declaration
public FastPolyline RemoveEquivalentAdjacentPoints(float minDistanceSquaredThreshold = 0.01)
Parameters
Type Name Description
float minDistanceSquaredThreshold
Returns
Type Description
FastPolyline
Remarks
  • The method always retains the first and last points of the polyline.
  • For polylines of lengths 2, the input polyline is returned.
  • For adjacent equivalent points, the first point is retained, except if one of the points is the last point of the polyline. In that case the last point is retained instead.
